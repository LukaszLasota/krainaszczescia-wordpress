## Production build
- install wordpress local
- copy database 
- copy and paste files from repository using git clone https://github.com/LukaszLasota/krainaszczescia-wordpress

## Webpack run
1\. Install the dependencies:

```
npm install
```
2\. Install the dependencies:

```
npm run dev
```
3\. Production build:

```
npm run build
```
